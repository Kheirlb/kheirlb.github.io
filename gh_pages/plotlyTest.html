CSV Online Plotter

Use the button below to select the local csv data file you would like to plot.

Keep in mind:

  • All data must be integers or floats.
  • Currently the header will be ignored until I figure out how to edit the axis/title/legend and shit.
  • First column of the csv is the x axis.
  • All remaining columns are various traces on the y axis.

A decent example csv file is included in the data folder of the repo.
